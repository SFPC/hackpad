Using Thunderbird & Enigmail 

How to set it up

  1. download 
    1. thunderbird & install (don’t set up your profile yet)
    2. download enaigmail plugin for thunderbird
  2. Open thunderbird - in ’Tools / Addons / Extensions manually add the plugin and restart
  3. Follow instructions here: https://www.enigmail.net/documentation/quickstart.php
  4. support site: http://support.gpgtools.org/kb/how-to/first-steps-where-do-i-start-where-do-i-begin
